Asking for Help

Do you have an issue with your code/project?

Always post your question in the relevant Coding Clinic channel. For example, a JavaScript question would go in the vanilla JavaScript channel.

If you're using a framework or library but we don't have a channel for it, just post in the most relevant vanilla channel. Wordpress, for example, would go in vanilla PHP.

The best way to get help is to ask your question while providing as many details and information as possible.

Tips for improving your question:

  • Don't ask if you can ask a question, just ask the original question
  • Provide as much code as possible. Use GitHub or Hastebin
  • Be clear about what you have tried, and errors you get
  • Try to list the version number of relevant tools and the OS
